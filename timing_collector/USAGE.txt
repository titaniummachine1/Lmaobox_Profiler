Lmaobox Profiler — timing_collector.exe
======================================

RUN
  Double-click timing_collector.exe (keep the window open).

DEPLOY LUA (once)
  Copy Profiler.lua to:  %LOCALAPPDATA%\lua\Profiler.lua
  (Developers: npm run bundle-deploy from the repo root.)

IN TF2
  lua_load simple_test        — one quick test
  lua_load multi_tick_test    — record while playing; unload to save

RESULT
  flame_graphs\<session_id>\tick.speedscope.json
  Open ONLY that file at https://www.speedscope.app
  (Do not open tick.folded.txt in speedscope.)

IF IT FAILS
  Game console shows: [Profiler] FAILED: <reason>
  Fix that message, then try again.
